- **H100/H200**: 完整支持所有优化
- **B200**: 需要 SM100 特定优化 (#12788, #17133)
- **A100/H800**: 部分优化适用

### 依赖关系
- **PyTorch**: >= 2.9.0
- **triton**: >= 3.4.0
- **sgl-kernel**: 需要最新版本
- **flashinfer**: >= 0.6.0
